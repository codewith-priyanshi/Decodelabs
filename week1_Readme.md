## Simple Rule-Based Chatbot

This project focuses on building a friendly, rule-based chatbot using Python. 
It uses conditional logic (if-else statements) to recognize user inputs, respond to greetings, answer basic predefined questions, 
and handle exits gracefully. This was a great exercise in understanding control flow and building the foundation for more complex conversational AI.

### Key Skills:

-Python basics and control flow.
-Handling user input through loops.
-Designing simple conversational logic.

## Technologies Used
- Python
